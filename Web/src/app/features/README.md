# Features

This directory contains the feature-specific logic of the application. Each feature should be self-contained.

## Structure per Feature
- **pages/**: Route-level components (Smart Components).
- **components/**: Feature-specific presentation components.
- **services/**: Services scoped to this feature (API calls for this domain).
- **models/**: Models specific to this feature.

## Features
- **auth/**: Login and Registration.
- **clients/**: Client management.
- **orders/**: Order management.
- **dashboard/**: Statistics and metrics.
